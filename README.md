# Cloning repo
```
git clone --depth 1 --recursive --shallow-submodules https://github.com/m09l6d0ur13ii/PandaMAP-Client.git
```

```bash
git submodule update --init --recursive
```

# Build cmake
```bash
mkdir build
cd build
cmake ..
make
```
